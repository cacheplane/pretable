# Support

## Questions

Use GitHub Discussions for usage questions, design discussion, and integration
help:

https://github.com/cacheplane/pretable/discussions

The public documentation is available at https://pretable.ai/docs/getting-started.

## Bugs

Use GitHub Issues for reproducible bugs. Include the smallest reproduction you
can provide, package versions, Node.js version, pnpm version, browser version
when relevant, and the command or app involved.

## Security

Do not file security issues publicly. Follow [SECURITY.md](./SECURITY.md).
